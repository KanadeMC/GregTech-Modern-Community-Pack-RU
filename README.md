# GregTech-Modern-Community-Pack-RU
Этот репозиторий был создан для того чтобы предоставить **любительский перевод** сборки всем желающим, так как подобных переводов в интернете для **новых** версий сборки я не нашёл.

## Инструкция по установке
> [!WARNING]
> **Перед установкой** Убедитесь в том что скачиваете подходящий вариант перевода - если у вас версия сборки ниже или ниже то что то может пойти не так.

Установка перевода осуществляется с помощью ручной замены файлов в папке сборки:
- Скачайте перевод ()
- Распакуйте в папку со сборкой, заменяя все предлагаемые файлы
